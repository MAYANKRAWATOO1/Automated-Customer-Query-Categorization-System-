# 🤖 AUTOMATED COUSTMER QUERY CATEGORIZATION SYSTEM

## 📌 Overview
This project is a Hybrid AI system combining Machine Learning, FAISS vector search, and Generative AI to provide intelligent customer support responses.

---

## 🚀 Features
- Query classification using ML models (SVM, LR, ANN, KMeans)
- Semantic search using FAISS
- AI-generated responses using LLM (LLaMA3 via Groq API)
- Interactive UI using Gradio

---

## 🧠 Technologies Used
- Python
- Scikit-learn
- FAISS
- Gradio
- Groq API (LLM)

---

## ⚙️ How It Works
1. User inputs query
2. ML model predicts category
3. FAISS finds most similar query
4. LLM generates final response
